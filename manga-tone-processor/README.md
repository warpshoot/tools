# 漫画背景風トーン処理

画像をアミ点トーンで漫画の背景風に変換するツールです。

## 機能

- 画像を4階調に自動分割（白/粗いトーン/細かいトーン/黒ベタ）
- トーン領域を円形ドットパターンで置き換え（45度回転）
- レベル補正機能（黒点・白点調整）
- PNG形式で出力

## 使い方

1. 画像をドラッグ&ドロップまたはクリックして選択（自動的に処理が実行されます）
2. スライダーで調整（リアルタイムに反映されます）
   - レベル補正：黒点・白点で明暗を調整
   - 粗いトーン：ドット間隔とサイズを調整
   - 細かいトーン：ドット間隔とサイズを調整
3. 「PNG出力」ボタンで画像をダウンロード
