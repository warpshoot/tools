# 漫画背景風トーン処理

画像をアミ点トーンで漫画の背景風に変換するツールです。

## 機能

- 画像を4つの明度レベルに自動分割
- 各レベルに18種類のトーンパターンから選択可能
  - 白（1種）
  - 粗いトーン（8段階）
  - 細かいトーン（8段階）
  - 黒ベタ（1種）
- 柔軟なトーン割り当て（白黒反転、粗密逆転など自由に設定可能）
- 輪郭線抽出機能（Sobelフィルタ）
  - ON/OFF切り替え
  - 感度調整（閾値スライダー）
  - トーンの上に黒線として重ねて表示
- 解像度固定機能（トーン密度の安定化）
- レベル補正機能（黒点・白点調整）
- PNG形式で出力

## 使い方

1. 画像をドラッグ&ドロップまたはクリックして選択（自動的に処理が実行されます）
2. 出力解像度を選択（1200px / 1600px / 2000px / オリジナル）
3. レベル補正で明暗を調整（黒点・白点スライダー）
4. 輪郭線抽出を設定（オプション）
   - チェックボックスでON/OFF切り替え
   - 感度スライダーで検出精度を調整
5. 各明度レベルにトーンパターンを割り当て
   - レベル1（最明部）〜レベル4（最暗部）
   - 各レベルで好きなパターンを選択
   - プレビューを見ながら調整
6. 「PNG出力」ボタンで画像をダウンロード

## トーン割り当ての例

- デフォルト: レベル1=白、レベル2=粗、レベル3=細、レベル4=黒
- 白黒反転: レベル1=黒、レベル2=細、レベル3=粗、レベル4=白
- 全トーン: レベル1=細1、レベル2=細3、レベル3=細5、レベル4=細7
