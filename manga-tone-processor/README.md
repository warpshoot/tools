# 漫画背景風トーン処理

画像をアミ点トーンで漫画の背景風に変換するツールです。

## 機能

- 画像を4つの明度レベルに自動分割
- 各レベルに18種類の多様なパターンから選択可能
  - 白（1種）
  - 粗いドット（4段階）
  - 細かいドット（4段階）
  - 斜線（3段階）
  - グリッド（3段階）
  - 有機的ドット（2段階）
  - 黒ベタ（1種）
- 柔軟なトーン割り当て（白黒反転、異なるパターンの組み合わせなど自由に設定可能）
- 輪郭線抽出機能（Sobelフィルタ）
  - ON/OFF切り替え
  - 感度調整（閾値スライダー）
  - トーンの上に黒線として重ねて表示
- 解像度固定機能（トーン密度の安定化）
- レベル補正機能（黒点・白点調整）
- 完全2値化出力（純粋な白黒画像）
- PNG形式で出力

## 使い方

1. 画像をドラッグ&ドロップまたはクリックして選択（自動的に処理が実行されます）
2. 出力解像度を選択（1200px / 1600px / 2000px / オリジナル）
3. レベル補正で明暗を調整（黒点・白点スライダー）
4. 輪郭線抽出を設定（オプション）
   - チェックボックスでON/OFF切り替え
   - 感度スライダーで検出精度を調整
5. 各明度レベルにトーンパターンを割り当て
   - レベル1（最明部）〜レベル4（最暗部）
   - 各レベルで好きなパターンを選択
   - プレビューを見ながら調整
6. 「PNG出力」ボタンで画像をダウンロード

## パターン割り当ての例

- デフォルト: レベル1=白、レベル2=粗ドット2、レベル3=細ドット2、レベル4=黒
- 白黒反転: レベル1=黒、レベル2=細ドット、レベル3=粗ドット、レベル4=白
- ライン調: レベル1=白、レベル2=斜線1、レベル3=斜線2、レベル4=黒
- グリッド調: レベル1=白、レベル2=グリッド1、レベル3=グリッド2、レベル4=黒
- 有機的: レベル1=白、レベル2=有機ドット1、レベル3=有機ドット2、レベル4=黒
- ミックス: レベル1=白、レベル2=斜線、レベル3=ドット、レベル4=グリッド
